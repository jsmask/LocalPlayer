# 本地音乐播放器

介绍：electron练习作，实现获取存储本地音频文件，原生js实现播放，window打包

![Image main](http://pic.yupoo.com/jsmask/ab9acd27/67f5fffc.png)

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn start
```

```
yarn dev
```

### Compiles and minifies for production
```
yarn dist
```
